AerodynamicObjects.AerodynamicObject Class Reference

An Aerodynamic Object determines the aerodynamic forces and moments acting on itself
due to the object's dimensions and state, as well as the state and properties of the fluid around the object. More...

Public Types

enum  ReferenceFrame
 The set of reference frames used in the aerodynamic models. Global (earth) frame is the inertial frame of reference. Local frame is the object's frame of reference. Body frame is a rotation from the object's frame to align (x, y, z) with (span, thickness, chord).

Detailed Description

An Aerodynamic Object determines the aerodynamic forces and moments acting on itself
due to the object's dimensions and state, as well as the state and properties of the fluid around the object.
